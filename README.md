## 3D Diel Utilization Distribution: a novel perspective to understand marine turtle-fishery interaction - R Code

This repository provides the R code used in the following research paper:

Javier Menéndez-Blázquez <a href="https://orcid.org/0000-0002-1882-7896"><img src="images/orcid.svg" alt="ORCID logo" width="21"/></a>, Jessica Ruff, Ricardo Sagarminaga, Brendan J. Godley, and David March.

------------------------------------------------------------------------

### Repository structure

| Folder | Description |
|----|----|
| `analysis/01_tracking` | Location processing, filtering, and movement regularization. |
| `analysis/02_3d_process` | Vertical habitat use calculation and 3D-MKDE model development. |
| `analysis/03_3d_fishing-effort` | Integration of fishing effort data and fishery-interaction analysis. |
| `analysis/04_habitat_use` | Statistical modeling of 3D habitat suitability. |
| `analysis/05_statistics_analysis` | Final statistical synthesis of home-range and overlap metrics. |
| `fig/` | R scripts and visualization pipelines for publication-quality figures. |
| `setup.R` | Environment configuration, package dependencies, and global parameters. |
